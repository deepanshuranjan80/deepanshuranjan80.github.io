Introduction to PHP

  • PHP का पूरा नाम Hypertext Preprocessor है। लेकिन इसको Personal Home Page भी कहते है.

  • PHP एक open source तथा server side scripting language है इसको HTML के साथ use में लाया जा सकता है.

  • इस language का इस्तेमाल website और application को बनाने के लिए किया जाता है। इसके साथ-साथ इसका प्रयोग dynamic web page create करने के लिए भी किया जाता है.

  • इसे वर्ष 1994 में Rasmus Lerdorf के द्वारा develop किया गया था और वर्ष 1995 में launch किया गया था.

  • PHP Language का प्रयोग web development में किया जाता है जिससे कि हम dynamic web page का निर्माण करते है तथा इसके द्वारा database, तथा e-commerce website को create किया जाता है.

  • PHP को हम free में download करके use कर सकते है.

  • PHP एक simple language है और इस language को सीखना काफी आसान है.

  • दूसरी scripting language की तुलना में PHP काफी fast language है। इस language को object-oriented के नाम से भी जाना जाता है। इस language को HTML में जोड़ा जा सकता है.

  • PHP लगभग सभी servers के साथ compatible होती है। अर्थात इसको सभी प्रकार के server के साथ काम में लाया जा सकता है.

  • PHP language को बहुत सारें platform में run कर सकते है जैसे:- Windows, Linux, Unix etc.

  • PHP file के .PHP, .PHP3, या .PHTML extension होते है.

How PHP work

  • जब कोई user browser में URL डालता है (जैसे `https://example.com/index.php`), तो यह web server को एक HTTP request भेजता है.

  • Server request को process करता है और देखता है कि file में `.php` extension है या नहीं.

  • यदि file PHP script वाली होती है, तो server इसे PHP interpreter को भेजता है.

  • PHP interpreter script को process करता है, जिसमें calculation, file access, या database से डेटा प्राप्त करना शामिल हो सकता है.

  • PHP कोड HTML के रूप में output देता है, जिसे browser में भेजने के लिए तैयार किया जाता है.

  • Process की गई HTML webpage के रूप में client (user के browser) को भेजी जाती है.

  • Browser प्राप्त किए गए HTML को display करता है, जिससे user को web page दिखता है.

  • PHP आवश्यकतानुसार MySQL, PostgreSQL, या अन्य database से data प्राप्त करता या store करता है.

  • PHP script client को नहीं दिखती, सिर्फ उसका output (HTML) ही user को मिलता है, जिससे security बनी रहती है.

  • PHP session (`$_SESSION`) और cookies (`$_COOKIE`) के माध्यम से user data को track कर सकता है.

  • PHP errors को handle कर सकता है और उन्हें log कर सकता है ताकि user को न दिखे, लेकिन developers इसे debug कर सके.

  • PHP, HTML, CSS, JavaScript और AJAX के साथ मिलकर काम कर सकता है.

  • PHP के लिए Laravel, CodeIgniter जैसे framework और WordPress, Joomla जैसे CMS उपलब्ध हैं.

  • PHP in build security features प्रदान करता है, जो SQL injection, XSS, और CSRF attacks को रोकते हैं.

  • PHP script को web hosting server पर upload किया जाता है, जिसमें PHP और Apache/Nginx जैसे web server install होते हैं.

  • PHP windows, linux और macOS servers पर चलता है और विभिन्न database और web server के साथ संगत है.

Features of PHP

  • PHP (Hypertext Preprocessor) एक server-side scripting language है जो web development के लिए उपयोग की जाती है. यह एक powerful language है जिसे dynamic और interactive web page बनाने के लिए design किया गया है. आइए PHP की Main Features (विशेषताओं) को विस्तार से समझते हैं:

1. Open Source & Free

  • PHP एक open source language है, जिसका अर्थ है कि इसे कोई भी free में download और use कर सकता है. इसके source code को आवश्यकतानुसार change किया जा सकता है.

2. Platform Independent

  • PHP को Windows, Linux, macOS आदि किसी भी operating system पर चलाया जा सकता है. यह विभिन्न servers जैसे Apache, Nginx, IIS आदि के साथ भी संगत है.

3. Server-Side Scripting

  • PHP एक server-side scripting language है, जिसका मतलब है कि इसका code server पर run होता है और output HTML के रूप में browser को भेजा जाता है. इससे website अधिक secure और fast बनती है.

4. Dynamic & Interactive Web Pages

  • PHP का उपयोग करके dynamic web page बनाए जा सकते हैं, जो user input के आधार पर content को बदल सकते हैं. जैसे – login system, form handling, e-commerce website आदि.

5. Easy to Learn & Use

  • PHP का syntax बहुत आसान और C, C++ व Java जैसी अन्य languages से मिलता-जुलता है. इसके कारण इसे सीखना और उपयोग करना आसान है.

6. Database Support

  • PHP विभिन्न database जैसे MySQL, PostgreSQL, Oracle, MongoDB आदि को support करता है, जिससे data को आसानी से store और manage किया जा सकता है.

7. High Performance & Speed

  • PHP अन्य scripting language की तुलना में fast है, क्योंकि यह server पर कम load डालता है और कम time में output generate करता है.

8. Security Features

  • PHP में कई security features शामिल हैं, जैसे – SQL Injection, Cross-Site Scripting (XSS), और Cross-Site Request Forgery (CSRF) से security के लिए in built function.

9. Large Community Support

  • PHP की एक large community है, जिससे कोई भी developer आसानी से सहायता प्राप्त कर सकता है। Online कई forms, tutorials और documentation उपलब्ध हैं.

10. Multiple Framework Support

  • PHP कई popular framework को support करता है, जैसे – Laravel, CodeIgniter, Symfony, Zend, Yii, CakePHP आदि, जो development को fast और simple बनाते हैं.

11. Object-Oriented Programming - OOPs

  • PHP में OOPs features मौजूद हैं, जिससे बड़े और modular projects को आसानी से develop किया जा सकता है.

12. File Handling

  • PHP में files को read, write, open, close करने और delete करने के लिए कई inbuilt functions होते हैं. यह features वेबसाइट पर file uploading और downloading में मदद करता है.

13. Multi-Threading Support

  • PHP एक समय में कई task को संभाल सकता है, जिससे यह high performance web application के लिए उपयुक्त बनता है.

14. Code Reusability

  • PHP में code को विभिन्न modules और functions में विभाजित किया जा सकता है, जिससे code को बार-बार उपयोग करना आसान हो जाता है.

15. Web Services & API Integration

  • PHP RESTful और SOAP web services को support करता है, जिससे अन्य application के साथ आसानी से data का आदान-प्रदान किया जा सकता है.
